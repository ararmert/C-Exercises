Aufgabe 2, 1 Punkt: Berechne die N-te Fibonacci Zahl. Alloziere dazu dynamisch ein Array und speichere darin das Ergebnis. Gebe das Array bzw. das Ergebnis zurück. Die Funktion wurde 10-Mal aufgerufen (10te Fibonacci Zahl).

Tipp 1: int array = malloc((n+1)*sizeof(int));

Tipp 2: array[0] = 0; array[1] = 1; for(int i = 2 ; i < N ; i++) array[i] = array[i-1] + array[i-2];
